Hello World

Raja Nagori is a Senior Information Security Engineer: IT Security Analyst II at FIS Global. He holds a bachelor's degree in computer science and engineering. He specializes in application security and penetration testing of web applications, network applications, and mobile applications (Android and iOS). He is currently working on Docker containerization and creating Docker images for Pentesters. Raja actively participates in the Hack the Box community and solves challenges that are completely based on new security patches. He also writes blogs about his security findings, methodologies for finding vulnerabilities, and mobile application VAPT methodologies on Medium Publications. He always shares his discoveries with the community to help the needy:)

OWASP Nightingale


About OWASP Nightingale

In today’s technological era, docker is the most powerful technology in each and every domain, whether it is Development, cyber security, DevOps, Automation, or Infrastructure. Considering the demand of the industry, I would like to introduce my idea to create a NIGHTINGALE: docker image for pentesters. This docker image is ready to use environment will the required tools that are needed at the time of pentesting on any of the scopes, whether it can be web application penetration testing, network penetration testing, mobile, API, OSINT, or Forensics. The best part is you can either create an altered docker image or pull the pre-built docker image from the hub. Some of the best features are listed below, I would highly recommend going through it and starting penetrating into the application.
